# CAMBIOS V19 - Campo, alertas y supervisión

## Mejoras incluidas
- Check-in con GPS obligatorio desde agenda diaria.
- Check-out con GPS obligatorio desde agenda diaria.
- Causales estructuradas para visitas no efectivas.
- Validación de evidencia obligatoria por causal sensible.
- Alertas automáticas por atraso o incumplimiento de agenda.
- Tablero de supervisión en tiempo real con auto-recarga.
- Corrección del pad de firma para tablet y lápiz táctil.

## Notas técnicas
- Se agregaron columnas nuevas a `visit_records` para GPS de check-in/check-out y alertas.
- Las alertas se notifican al funcionario si la visita entra en atraso.
- La firma ahora conserva mejor el tamaño del canvas y el pointer capture.
- La agenda diaria incluye acciones rápidas para check-in/check-out y actualización con causal.
