# MANUAL OPERATIVO Y GUÍA DE USUARIO
**GeoBusca Territorial**

## 1. FLUJO DE TRABAJO
1.  **Carga:** Sube archivos CSV o Excel.
2.  **Limpieza:** Deduplicación automática contra el historial.
3.  **Proceso:** Geocodificación masiva con barra de progreso.
4.  **Cruce:** Asignación automática de Barrio/Vereda.

## 2. REVISIÓN Y MAPA
*   Visualiza puntos con "Confianza Baja".
*   Edición manual mediante arrastrar y soltar (Drag & Drop) en el mapa.

## 3. GESTIÓN DE CAMPO
*   Asignación de visitas a funcionarios.
*   Captura de evidencias fotográficas y GPS en tiempo real.
*   Sincronización automática de datos recolectados.

## 4. EXPORTACIÓN
Formatos soportados: Excel, CSV, GeoJSON y KML.
